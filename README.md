# Media Molecule Test

## Demo

https://lnesi.github.io/mmtest/index.html

### Install

```
git clone https://github.com/lnesi/mmtest.git
```

#### Requirements
Node +9
```
npm install
```

### Development
Start local devleopment server
```
npm run start
```
### Build
Build compile version for host
```
npm run build
```
### Deploy

Deploy to Github Pages

```
npm run deploy
```
### Unit Testing

```
npm run test
```
